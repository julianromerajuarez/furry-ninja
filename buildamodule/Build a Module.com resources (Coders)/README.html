Build a Module.com Resource Pack - v.26

This resource pack contains the source files for Build a Module.com videos. Below is a list of the modules included in this pack - in alphabetical order - with a brief description of each. Each folder represents a module, and each module includes a folder called steps. The steps folder contains complete files whose source code can be copied over during particular parts of a video in order to follow along.

Note that any corrections to the videos will be mentioned in the corresponding section of the source code.

Enjoy!

  • assets - The assets module contains examples used to demonstrate how to work with files and the File API.
  • input - Explores using the Form API to create forms and dynamically generate new form elements.
  • interact - Used in the jQuery and JavaScript videos to demonstrate the basic use of jQuery and writing JavaScript for Drupal.
  • lookandfeel - Demonstrates how to use the theming layer within a module.
  • pages - Explores how to use the menu system to create new pages and page callbacks.
  • persons - A module that demonstrates how to work with users in module code.
  • scaffolding - A basic module structure that can be used as a basis for any module.
  • stuff - Explores how to work with nodes and node permissions in a module.
  • tips - A companion module to the Development Tips and Tricks videos.
  • trails-01-hooks - Explores how to use the hook system to create a block which tracks user history.
  • trails-02-database-api - Building on the previous "trails" module, this demonstrates how to use the Database API to create, edit, delete and modify table records, as well as use the Schema API to create tables initially, all while creating a practical module to track user history.
  • twominutes - A phrase invoked by toddlers everywhere, this is also a module that's created in the initial videos to demonstrate how quickly one can build a module.
